1. 安装或更新 Chrome 浏览器 2. 开启开发者模式 3. 导入并启用插件 4. 验证安装效果

1安装或更新 Chrome 浏览器

插件依赖桌面版 Chrome 浏览器。请确认您使用的是最新版，以便顺利开启开发者模式并加载扩展。

操作步骤：
  • 访问 Google 官方下载站：https://www.google.cn/chrome/
  • 下载后按照系统提示完成安装；若已安装，请进入 设置 > 关于 Chrome 检查并更新到最新版本。
  • 国内网络环境若无法访问，可从各大应用商店或者软件站下载安装包。

2开启 Chrome 开发者模式

要加载本地扩展，必须先打开 Chrome 扩展管理界面的开发者模式。

建议操作：
  • 在地址栏输入 chrome://extensions/ 并按回车。
  • 在页面右上角找到“开发者模式”开关，将其打开。
  • 若看不到开关，可通过 菜单 > 更多工具 > 扩展程序 打开同一页面。
打开开发者模式后，页面右上角会显示 “加载已解压的扩展程序” 按钮，这就是后续导入插件的入口。

3导入并启用插件

从压缩包中解压到电脑本地，随后通过扩展程序页面加载。

导入步骤：
  • 将下载得到的 aifa-extension.zip 解压到一个易于记忆的位置，例如桌面或文档目录。
  • 在 chrome://extensions/ 页面点击 “加载已解压的扩展程序”。
  • 在弹出的文件夹选择窗口中，指定刚解压的 aifa-extension 文件夹，然后确认。
  • 导入完成后，确保扩展卡片右下角的开关处于开启状态。

4验证安装效果

完成导入后，请按以下方法验证扩展是否已正确启用。

检查要点：
  • Chrome 地址栏右侧出现 “AIFA 发布助手” 图标，点击后能看到版本号与快捷说明。
若以上未通过，请重新检查开发者模式是否开启、扩展开关是否打开，或通过浏览器右上角扩展图标选择“管理扩展程序”重新加载。
